Kuler

Kuler, a product created by Adobe to make life easier for web designers. It’s a type of software where you can create color palettes for designs and even explore other palettes that other people in the community have made. Recently Adobe has created an application that works hand-in-hand with Kuler for mobile devices where the application takes the most prominent colors of an image you capture and creates a pallet from it which you can also save and let other people explore.

Check out Kuler today!

CSS-Tricks

CSS-Tricks – "A community curated by Chris Coyier." CSS-Tricks is a web design resource where you can learn about Complex CSS3 techniques. You can learn code through his fantastic, easy to understand, sometimes very humorous articles, the Extensively informative and in-depth videos, through the Extremely welcoming and helpful forums, through his code snippets sections or check out a demo from the demo page.

Check out CSS-Tricks!

W3C

The accuracy and semantics of your online, published code is imperative for it to be valid. Not only for functionality, but also future proofing. While technology changes daily with new trends, analytics resources, Its important to get your code available online. W3C can set you on the straight an narrow by validating your code, whether it's HTML5, CSS3 and Javascript to much more robust types of coding languages such as Objective C, Java and Ruby.

Check out W3C.org today!
